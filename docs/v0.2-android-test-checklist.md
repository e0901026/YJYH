# V0.2 Android 测试清单

> 范围：Android 工程壳、主题、页面路由、底部导航、mock 页面。

## 1. 构建检查

- `gradle assembleDebug` 可执行。
- `gradle testDebugUnitTest` 可执行。
- `gradle lintDebug` 可执行。
- 若本机缺 Android SDK，需要记录为环境问题，不视为产品功能失败。
- 当前 Codex 环境已发现 Java / Gradle / Android SDK 缺失；需要先补齐工具链后才能执行构建命令。

## 2. 启动检查

- App 可安装。
- App 可启动到登录页。
- 登录页展示工号、密码、登录、使用邀请码注册。

## 3. 页面导航

- 登录后进入首页。
- 注册页可以从登录页进入。
- 首页底部导航显示：首页、设备、我的。
- 设备页底部导航选中设备。
- 我的页底部导航选中我的。
- 扫码借、手机注册、一键还、设备详情、Owner 页面不显示底部导航。

## 4. 原型还原

- 页面背景为浅灰。
- 卡片为白底浅描边。
- 主按钮为蓝底白字。
- 次按钮为白底蓝描边。
- 设备列表筛选为 Tab，而不是主按钮。
- Owner 用户列表 / 邀请码为页面顶部 Tab。

## 5. V0.2 不验收

- 不验收真实登录。
- 不验收真实扫码。
- 不验收真实后端数据。
- 不验收真实通知推送。
